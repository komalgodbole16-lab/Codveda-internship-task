# Responsive Landing Page

## Description
This project is a responsive landing page created using HTML, CSS, and JavaScript. 
The landing page is designed to adapt smoothly to different screen sizes such as 
mobile, tablet, and desktop. It follows clean UI principles and a structured layout 
to provide a better user experience.

## Features
- Fully responsive design
- Mobile-first approach
- Clean and modern UI
- Cross-device compatibility

## Technologies Used
- HTML
- CSS
- JavaScript

## Objective
To demonstrate responsive web design skills and the ability to create visually 
appealing landing pages using frontend technologies.
