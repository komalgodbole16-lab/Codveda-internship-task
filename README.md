# Frontend Internship Tasks

This repository contains three frontend tasks completed as part of a frontend internship. Each task demonstrates different skills in HTML, CSS, and JavaScript.

---

## Task 1: Responsive Landing Page

**Description:**  
This task focuses on building a fully responsive landing page that adapts smoothly to different screen sizes including mobile, tablet, and desktop. A clean layout and user-friendly design principles are followed.

**Key Features:**
- Fully responsive design  
- Mobile-first layout  
- Clean UI and proper spacing  
- HTML, CSS, and JavaScript used  

---

## Task 2: Form Validation using JavaScript

**Description:**  
This task involves creating an interactive form that validates user input using JavaScript. The form includes fields such as name, email, phone number, and password. Real-time validation provides instant feedback to users.

**Key Features:**
- Real-time input validation  
- Error messages without page reload  
- Email format and password strength validation  
- Improved user experience using JavaScript
